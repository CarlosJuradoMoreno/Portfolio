# Carlos Jurado Moreno — Portfolio

Premium personal portfolio for a Senior Software Engineer focused on enterprise backend systems, Java, Spring Boot, and cloud-native technologies.

## Overview

The website presents professional experience, technical skills, engineering principles, selected work, open-source activity, and AI-related experiments through a responsive dark-mode interface.

Key features:

- Responsive, mobile-first layout
- English, German, and Spanish language selector
- Persisted language preference
- Animated sections and scroll reveals
- Enterprise Billing Platform project showcase
- GitHub and LinkedIn integration
- SEO and Open Graph metadata
- Reduced-motion accessibility support
- Optimized local hero artwork

## Technology stack

- Next.js 15 with App Router
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Radix UI primitives
- shadcn/ui-style components
- Lucide icons

## Getting started

Install the dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Production

Create an optimized production build:

```bash
npm run build
```

Run the production server:

```bash
npm start
```

## Project structure

```text
app/
  globals.css        Global styles and design tokens
  layout.tsx         Root layout and SEO metadata
  page.tsx           Home page entry point
components/
  portfolio.tsx      Portfolio sections and interactions
  ui/button.tsx      Reusable button component
lib/
  i18n.ts            English, German, and Spanish translations
  utils.ts           Shared utility functions
public/
  hero-system.png    Hero artwork
```
